## Dice
Formatted Dice Roller in Python3

## Usage
<pre>
$ <b>python3 dice.py</b>
Roll Dice: <b>1d6</b>
1d6+0
[6]
6

Roll Dice: <b>4d4+2 2d10-8</b>
4d4+2
[4, 4, 3, 2]
15

2d10-8
[8, 10]
10

Roll Dice: <b>quit</b>
exiting
</pre>

## Installation

1. Install Python 3
2. `python3 dice.py`
