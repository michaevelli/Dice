## Dice
Formatted Dice Roller in Python3

## Usage
```bash
$ python3 dice.py
Roll Dice: 1d6
1d6+0
[6]
6

Roll Dice: 4d4+2 2d10-8
4d4+2
[4, 4, 3, 2]
15

2d10-8
[8, 10]
10

Roll Dice: quit
exiting
```

## Installation

1. Install Python 3
2. `python3 dice.py`
